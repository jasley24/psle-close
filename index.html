Set up Cloud Scores

Scores sync across all devices in real time using Firebase (free).

5-minute setup:
1. Go to console.firebase.google.com
2. Click Add project → name it psle-cloze → Create
3. Left menu → Build → Realtime Database
4. Click Create Database → any region → Start in test mode → Enable
5. Copy the URL shown (like https://psle-cloze-xxxxx-default-rtdb.firebaseio.com)
6. Paste it below
